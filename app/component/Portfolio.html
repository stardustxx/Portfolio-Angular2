Hey

Remind people of what they need to do

Using Hey to send a remind your closed one’s phone to send message back to you when they, for example, arrive at their office or when it’s certain time. Reminder’s preliminary requirement is the user needs to know what task at when the user needs to be doing. With this app, one can send a message to another user and set a scenario as to when this message should show up. Instant messaging usually require minimum attention so if there is task, people sometimes just forget about it afterwards if one never remembers to set a reminder. This app would allow one to reminder another one at the right time. It can be used between closed ones, or a group of team for productivity.
